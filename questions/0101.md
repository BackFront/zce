Consider the following PHP script:
```php
$a = 5;
$b = 10;
function Mul() {
    $a =0;
    $b = $a * $b;
}
Mul();
print($b);
```
What will be the output of the above script?

- [ ] A) 10
- [ ] B) The script will return an error message.
- [ ] C) 0
- [ ] D) 50

<details><summary><b>Answer</b></summary>
<p>
  Answer: <strong>A</strong>
</p>
