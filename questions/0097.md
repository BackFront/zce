Which of the following statements will return the second parameter passed to a function?

- [ ] A)
```php
func_get_arg(1);
```

- [ ] B)
```php
func_get_args(2);
```

- [ ] C)
```php
func_num_args(2);
```

- [ ] D)
```php
func_num_args(1);
```


<details><summary><b>Answer</b></summary>
<p>
  Answer: <strong>A</strong>
</p>
</details>
